# SecondWear 👗

A React marketplace for buying and selling second-hand clothing.
Built with React, Vite, and React Router.

## Features
- Browse curated pre-loved fashion items
- Product detail pages with full descriptions
- Shopping cart with add/remove functionality
- Sell your own items via the listing form
- Responsive design with a warm cream & brown aesthetic

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack
- React 19
- Vite
- React Router DOM
- Lucide React (icons)
